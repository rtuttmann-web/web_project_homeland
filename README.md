# De Pátria para Pátria

## Descrição

Página web responsiva que apresenta as cidades natais de colaboradores da
TripleTen, com histórias, fotografias e a possibilidade de adquirir obras
de arte digitais como NFT.

As cidades apresentadas são: Kyiv (Ucrânia), Criccieth (País de Gales),
Berea (EUA), Muramvya (Burundi) e Braga (Portugal).

## Tecnologias e técnicas utilizadas

- HTML5 semântico (`<header>`, `<main>`, `<footer>`, `<section>`, `<article>`)
- CSS3 com Flexbox e CSS Grid para layout
- Design responsivo com Media Queries (breakpoints: 544px e 1024px)
- Metodologia BEM com estrutura de ficheiros BEM Flat
- Fonte Inter carregada localmente via `@font-face`
- `object-fit` para controlo de imagens
- `normalize.css` para consistência entre browsers
